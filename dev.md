set username & email 
	git config --global user.name "YOURNAME"
	git config --global user.email "your@email.com"
----------------------
Check status
	git status
----------------------
Add into git
	git add . : add all file
	git add '*.txt' : add about file .txt
----------------------
comment when you edit
	git commit -m "add sample index page"
----------------------
See all branch
	git branch
----------------------
change branch
	git checkout
----------------------
bring file together
	git merge
